# Usage
Enter `FineLib.setPlugin(this)` in `onEnable()` of the plugin to be used.
```java
public class Your_Plugin extends JavaPlugin{
    @Override
    public void onEnable(){
        FineLib.setPlugin(this);
    }
    // your code...
}
```
This library is for **Minecraft 1.18**
